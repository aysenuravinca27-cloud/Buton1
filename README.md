
 Buton1

Bu proje, HTML ve CSS kullanılarak geliştirilmiş basit bir buton tasarımı çalışmasıdır. Projede temel HTML yapısı, CSS ile stil düzenlemeleri ve hover efekti kullanılarak kullanıcı etkileşimi oluşturulmuştur.

 ✨ Özellikler
- Temiz ve sade buton tasarımı
- CSS ile renk, boyut ve hizalama düzenlemeleri
- Hover efekti ile kullanıcı etkileşimi
- Basit ve anlaşılır dosya yapısı

🛠️ Kullanılan Teknolojiler
- HTML
- CSS

📁 Proje Yapısı
- `index.html` → Sayfanın temel HTML yapısı
- `style.css` → Buton tasarımı ve stil düzenlemeleri

 🚀 Nasıl Çalıştırılır?
Projeyi indirdikten sonra `index.html` dosyasını tarayıcıda açabilirsiniz.
